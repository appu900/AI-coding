# ⚡ DevPath AI — AI-Powered Coding Learning Platform

A full-stack Next.js 14 application that uses Claude AI to assess students, generate personalized learning roadmaps, and create course content.

---

## Tech Stack

| Layer       | Technology                  |
|-------------|-----------------------------|
| Framework   | Next.js 14 (App Router)     |
| Auth        | NextAuth.js (Credentials)   |
| Database    | MongoDB + Mongoose           |
| AI          | Anthropic Claude (claude-opus-4-5) |
| Styling     | Tailwind CSS                |
| Language    | TypeScript                  |

---

## Features

### Student Flow
1. **Sign up** → email + password registration
2. **Browse courses** → MongoDB, JavaScript, React (admin-configured)
3. **AI Assessment** → 10 MCQ questions generated by Claude, adapted to the course
4. **Level detection** → Beginner / Intermediate / Advanced based on score
5. **Personalized roadmap** → AI-generated 0-to-hero learning path tailored to weak/strong areas
6. **Progress tracking** → Mark steps complete, visual progress bars per roadmap

### Admin Flow
1. **Create courses** → title, description, icon, color, tags
2. **Publish / unpublish** → control student visibility
3. **Add modules** → manually or trigger AI content generation (Claude generates submodules with explanations, code examples, exercises)
4. **View students** → see every student's level, test score, and roadmap progress

## Quick Start

### 1. Clone & Install

```bash
git clone <repo-url>
cd ai-coding-platform
npm install
```

### 2. Environment Variables

```bash
cp .env.example .env.local
```

Edit `.env.local`:

```env
# MongoDB (local or Atlas)
MONGODB_URI=mongodb://localhost:27017/ai-coding-platform

# NextAuth
NEXTAUTH_SECRET=any-random-32-char-string
NEXTAUTH_URL=http://localhost:3000

# Anthropic
ANTHROPIC_API_KEY=sk-ant-...

# First admin account (this email gets admin role automatically)
ADMIN_EMAIL=admin@yoursite.com
```

### 3. Seed Initial Courses (optional)

After starting, register with your `ADMIN_EMAIL` then create courses from the admin dashboard, or run the seed script:

```bash
# Start MongoDB locally (if not using Atlas)
mongod

# Run the app
npm run dev
```

### 4. Open

```
http://localhost:3000
```

Register using the `ADMIN_EMAIL` → you'll get admin access automatically.

## AI Capabilities (Claude API)

All AI calls use `claude-opus-4-5`.

| Function                    | Description                                      |
|-----------------------------|--------------------------------------------------|
| `generateMCQTest()`         | 10 questions mixing beginner/intermediate/advanced |
| `analyzeTestResults()`      | Scores answers, determines level, finds weak/strong areas |
| `generatePersonalizedRoadmap()` | Creates 8–12 ordered steps tailored to student level |
| `generateModuleContent()`   | Generates submodules with content, code, exercises |

## Adding More Courses

As admin:
1. Go to `/dashboard/admin/courses`
2. Click "New Course"
3. Fill in title, description, icon (emoji), color
4. Click "Create Course"
5. Expand the course → add modules (tick "AI generate content" for automatic submodule generation)
6. Click the eye icon to publish the course so students can see it
